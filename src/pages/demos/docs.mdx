# Ravin セットアップガイド <Label type="info">v0.1</Label>

このドキュメントでは、Ravinテーマのインストールから初期設定までの手順を解説します。

<Callout type="warning" title="Warning">
  現在提供しているのはベータ版です。本番環境での利用の際は、事前に十分なテストを行ってください。
</Callout>

## 1. インストール手順

お使いのパッケージマネージャーに合わせて、以下のコマンドを実行してください。

<Tabs labels={['npm', 'pnpm', 'yarn']}>
  <div>
    <CodeBlock>
    ```bash
    npm install ravin
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```bash
    pnpm add ravin
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```bash
    yarn add ravin
    ```
    </CodeBlock>
  </div>
</Tabs>

## 2. 初期セットアップ

<Steps>
1. **プロジェクトの初期化**

   Astroプロジェクトを作成し、ディレクトリに移動します。
2. **設定ファイルの編集**

   `astro.config.mjs` を開き、Ravinをインテグレーションに追加します。
3. **開発サーバーの起動**

   `npm run dev` でローカルサーバーを起動し、表示を確認します。
</Steps>

## 3. プロパティ一覧

ドキュメント内でAPIリファレンスなどを記述する際は、Tableコンポーネントを使用します。

<Table>
| プロパティ | 型 | 必須 | 説明 |
| :--- | :--- | :--- | :--- |
| `title` | `string` | 〇 | 記事またはドキュメントのタイトル |
| `date` | `Date` | △ | 公開日（ブログ用途で推奨） |
| `tags` | `string[]` | △ | 関連するタグの配列 |
</Table>
