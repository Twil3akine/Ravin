# Button

リンクやアクションを促すためのボタンコンポーネントです。

## Properties

Buttonコンポーネントは以下のプロパティを受け付けます。

<Table>
  | Property | Type | Description |
  | :------- | :--- | :---------- |
  | `type` | `"primary" \| "default"` | ボタンのスタイルを選択します。 |
  | `href` | `string` | リンク先URL。指定すると `<a>`、指定しないと `<button>` タグになります。 |
  | `square` | `boolean` | `true` の場合、正方形の形状になります（アイコン配置などに最適）。 |
</Table>

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div style="display: flex; gap: 1rem; flex-wrap: wrap; margin-bottom: 1.5rem; align-items: center;">

    <Button href="#" type="default">Default</Button>
    <Button href="#" type="default" square={true}>

    </Button>
