# Avatar

ユーザーのプロフィール画像やイニシャルを表示するためのコンポーネントです。

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div>
    <h3 style="margin-top: 0">画像を使用する場合</h3>
    <div style={{ display: 'flex', gap: '1rem', alignItems: 'center', marginBottom: '1.5rem' }}>
      <Avatar src="https://avatars.githubusercontent.com/u/9919?v=4" alt="GitHub" size="lg" />
      <Avatar src="https://avatars.githubusercontent.com/u/9919?v=4" alt="GitHub" size="md" />
      <Avatar src="https://avatars.githubusercontent.com/u/9919?v=4" alt="GitHub" size="sm" />
    </div>

    <h3>イニシャルを使用する場合（フォールバック）</h3>
    <div style={{ display: 'flex', gap: '1rem', alignItems: 'center', marginTop: '1rem', marginBottom: '1.5rem' }}>
      <Avatar initials="Ravin" size="lg" />
      <Avatar initials="Ravin" size="md" />
      <Avatar initials="Ravin" size="sm" />
    </div>
  </div>
  <div>
    ```html
    <!-- 画像を使用する場合 -->
    <Avatar src="https://avatars.githubusercontent.com/u/9919?v=4" alt="GitHub" size="lg" />
    <Avatar src="..." alt="..." size="md" />
    <Avatar src="..." alt="..." size="sm" />

    <!-- イニシャルを使用する場合（フォールバック） -->
    <Avatar initials="Ravin" size="lg" />
    <Avatar initials="Ravin" size="md" />
    <Avatar initials="Ravin" size="sm" />
    ```
  </div>
</Tabs>
