# Table

表形式のデータを表示するためのコンポーネントです。
画面幅が狭い場合は横スクロールが可能になります。

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div style="margin-bottom: 1.5rem;">
    <Table>
    | プロパティ | 型 | 説明 |
    | :--- | :--- | :--- |
    | `title` | `string` | コンポーネントのタイトルを指定します。 |
    | `href` | `string` | リンク先のURLを指定します。 |
    | `disabled` | `boolean` | 無効状態にするかどうかを指定します。 |
    </Table>
  </div>
