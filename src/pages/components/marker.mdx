# Marker

テキストの一部を蛍光ペンで引いたように強調するインライン要素です。

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div style="display: flex; flex-direction: column; gap: 1rem; margin-bottom: 1.5rem;">
    <p>重要な箇所を <Marker type="warning">このように強調</Marker> することができます。</p>
    <p><Marker type="info">Info</Marker>、<Marker type="success">Success</Marker>、<Marker type="danger">Danger</Marker> も使用可能です。</p>
  </div>
  <div>
    ```html
    重要な箇所を <Marker type="warning">このように強調</Marker> することができます。

    ```
  </div>
