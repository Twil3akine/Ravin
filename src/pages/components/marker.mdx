# Marker

テキストの一部を蛍光ペンで引いたように強調するインライン要素です。

## Usage

重要な箇所を <Marker type="warning">このように強調</Marker> することができます。<Marker type="info">Info</Marker>、<Marker type="success">Success</Marker>、<Marker type="danger">Danger</Marker> も使用可能です。
