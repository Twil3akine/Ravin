# Toggle

ON/OFFの状態を切り替えるためのシンプルなスイッチコンポーネントです。

## Usage

<Callout type="note" title="Note">
  このコンポーネントはSvelteで構築されています。MDX内などでインタラクティブに動作させるには `client:load` などのクライアントディレクティブが必要です。
</Callout>

<Tabs labels={["Preview", "Code"]}>
  <div>
    <h3 style="margin-top: 0">デフォルト（OFF）</h3>
    <div style="margin-bottom: 1.5rem;">
      <Toggle client:load />
    </div>

    <h3>初期状態がON</h3>
    <p><code>checked={'{true}'}</code> を渡すことで、初期状態をONにできます。</p>
    <div style="margin-bottom: 1.5rem;">
      <Toggle checked={true} client:load />
    </div>

    <h3>無効化（Disabled）</h3>
    <p><code>disabled={'{true}'}</code> を渡すと、クリックできない状態になります。</p>
    <div style="display: flex; gap: 1rem; margin-bottom: 1.5rem;">
      <Toggle disabled={true} client:load />
      <Toggle checked={true} disabled={true} client:load />
    </div>
  </div>
  <div>
    ```html
    <!-- デフォルト（OFF） -->
    <Toggle client:load />

    <!-- 初期状態がON -->
    <Toggle checked={true} client:load />

    <!-- 無効化（Disabled） -->
    <Toggle disabled={true} client:load />
    <Toggle checked={true} disabled={true} client:load />
    ```
  </div>
</Tabs>
