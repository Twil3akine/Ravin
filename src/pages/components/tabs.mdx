# Tabs

OSごとの手順や、コードとプレビューの切り替えなど、コンテンツをタブで分けて表示するためのコンポーネントです。

## Usage

  ### 1. パッケージマネージャーの切り替え

  <Tabs labels={['npm', 'pnpm', 'bun']}>
    <div>
      <CodeBlock>
      ```bash
      npm install ravin
      ```
      </CodeBlock>
    </div>
    
    <div>
      <CodeBlock>
      ```bash
      pnpm add ravin
      ```
      </CodeBlock>
    </div>
    
    <div>
      <CodeBlock>
        ```bash
        bun add ravin
        ```
      </CodeBlock>
    </div>
  </Tabs>

  ### 2. フレームワーク別のコード比較

  <Tabs labels={['React', 'Vue', 'Svelte']}>
    <div>
      <CodeBlock>
    ```jsx
    function Hello({ name }) {
      return <h1>Hello, {name}!</h1>;
    }
    ```
      </CodeBlock>
    </div>
    
    <div>
      <CodeBlock>
      ```vue
      <template>
        <h1>Hello, {{ name }}!</h1>
      </template>
      <script setup>
      const props = defineProps(['name']);
      </script>
      ```
      </CodeBlock>
    </div>
    
    <div>
      <CodeBlock>
      ```svelte
      <script>
        export let name;
      </script>

      <h1>Hello, {name}!</h1>
      ```
      </CodeBlock>
    </div>
  </Tabs>

  ### 3. Code & Preview の切り替え

  コンポーネントのデモ用途などで、プレビュー画面とソースコードをタブで分けることができます。

  <Tabs labels={['Preview', 'Code']}>
    <div>
      <div style="padding: 2rem; border: 1px dashed var(--theme-border); border-radius: 6px; margin-top: 1.5rem; display: flex; justify-content: center;">
        <Button href="#" type="primary">プレビューボタン</Button>
      </div>
    </div>
    
    <div>
      <CodeBlock>
      ```astro
      ---
      import Button from '@/components/Button.astro';
      ---

      <Button href="#" type="primary">プレビューボタン</Button>
      ```
      </CodeBlock>
    </div>
  </Tabs>
