# Tabs

OSごとの手順や、コードとプレビューの切り替えなど、コンテンツをタブで分けて表示するためのコンポーネントです。

## Usage

  ### 1. パッケージマネージャーの切り替え

  <Tabs labels={['npm', 'pnpm', 'yarn']}>
    <Fragment>
      <CodeBlock>
      ```bash
      npm install ravin
      ```
      </CodeBlock>
    </Fragment>
    <Fragment>
      <CodeBlock>
      ```bash
      pnpm add ravin
      ```
      </CodeBlock>
    </Fragment>
    <Fragment>
      <CodeBlock>
      ```bash
      yarn add ravin
      ```
      </CodeBlock>
    </Fragment>
  </Tabs>

  ### 2. OS別の手順の切り替え

  <Tabs labels={['macOS / Linux', 'Windows']}>
    <Fragment>
      ターミナルを開き、以下のコマンドを実行します。
      <CodeBlock>
      ```bash
      ls -la
      ```
      </CodeBlock>
    </Fragment>
    <Fragment>
      PowerShellを開き、以下のコマンドを実行します。
      <CodeBlock>
      ```powershell
      Get-ChildItem -Force
      ```
      </CodeBlock>
    </Fragment>
  </Tabs>
