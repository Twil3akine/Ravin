# Tabs

OSごとの手順や、コードとプレビューの切り替えなど、コンテンツをタブで分けて表示するためのコンポーネントです。

## Usage

### 1. パッケージマネージャーの切り替え
ドキュメントで頻出する、インストールコマンドの切り替え例です。

<Tabs labels={['npm', 'pnpm', 'yarn']}>
  <div>
    <CodeBlock>
    ```bash
    npm install ravin
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```bash
    pnpm add ravin
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```bash
    yarn add ravin
    ```
    </CodeBlock>
  </div>
</Tabs>

### 2. OS別の手順の切り替え
環境によって異なる手順や説明を案内する例です。Markdownのテキストとコンポーネントを混在させることができます。

<Tabs labels={['macOS / Linux', 'Windows']}>
  <div>
    ターミナルを開き、以下のコマンドを実行します。
    <CodeBlock>
    ```bash
    ls -la
    ```
    </CodeBlock>
  </div>
  <div>
    PowerShellを開き、以下のコマンドを実行します。
    <CodeBlock>
    ```powershell
    Get-ChildItem -Force
    ```
    </CodeBlock>
  </div>
</Tabs>

### 3. プレビューとコードの切り替え
実際のコンポーネントの見た目と、その実装コードを切り替えて表示する例です。

    ```astro
    <Button href="#" type="primary">プレビューボタン</Button>
    ```
    </CodeBlock>
  </div>
</Tabs>
