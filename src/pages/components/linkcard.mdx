# LinkCard

指定したURLのOGP（タイトル、説明、サムネイル画像）をビルド時に自動取得し、リッチなカード形式で表示するAstroコンポーネントです。

## Usage

MDX内で `<LinkCard href="..." />` のように記述するだけで、美しいカードが生成されます。

<Tabs labels={["Preview", "Code"]}>
  <div style="display: flex; flex-direction: column; gap: 1.5rem; margin-bottom: 1.5rem;">
    <div>
      <h3 style="margin-top: 0">Astro 公式サイトの例</h3>
      <LinkCard href="https://astro.build/" />
    </div>
    <div>
      <h3 style="margin-top: 0">GitHub の例</h3>
      <LinkCard href="https://github.com/" />
    </div>
  </div>
  <div>
    ```mdx
    ### Astro 公式サイトの例

    <LinkCard href="https://astro.build/" />

    ### GitHub の例

    <LinkCard href="https://github.com/" />
    ```
  </div>
</Tabs>
