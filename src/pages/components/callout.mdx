# Callout

補足情報や警告を視覚的に伝えるためのコンポーネントです。

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div style="display: flex; flex-direction: column; gap: 1rem; margin-bottom: 1.5rem;">
    <Callout type="note" title="Note">標準的な補足情報です。</Callout>
    <Callout type="tip" title="Tip">役立つヒントやベストプラクティスです。</Callout>
    <Callout type="warning" title="Warning">注意が必要な情報です。</Callout>
    <Callout type="danger" title="Danger">非推奨や破壊的な変更を示す警告です。</Callout>
  </div>
