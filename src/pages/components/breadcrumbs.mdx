# Breadcrumbs

現在のページの階層構造を示し、ナビゲーションを助けるコンポーネントです。

## Usage

<Tabs labels={["Preview", "Code"]}>
  <div>
    <h3 style="margin-top: 0">自動生成</h3>
    <p>プロパティを渡さない場合、現在のURLから自動的にパンくずリストが生成されます。</p>
    <Breadcrumbs />

    <h3 style="margin-top: 1.5rem">手動指定</h3>
    <p>任意の階層をプロパティとして渡すことも可能です。</p>
    <div style="margin-bottom: 1.5rem;">
      <Breadcrumbs
        crumbs={[
          { label: "Home", href: "/" },
          { label: "Components", href: "/components" },
          { label: "Breadcrumbs" },
        ]}
      />
    </div>
  </div>
  <div>
    ```html
    <h3>自動生成</h3>
    <p>プロパティを渡さない場合、現在のURLから自動的にパンくずリストが生成されます。</p>
    <Breadcrumbs />

    <h3>手動指定</h3>
    <p>任意の階層をプロパティとして渡すことも可能です。</p>
    <Breadcrumbs
      crumbs={[
        { label: "Home", href: "/" },
        { label: "Components", href: "/components" },
        { label: "Breadcrumbs" },
      ]}
    />
    ```
  </div>
</Tabs>
