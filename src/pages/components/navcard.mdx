# NavCard

内部ページへの導線や、目次リストを構築するためのAstroコンポーネントです。

## Usage

基本的には `<ul>` などのリスト要素の中で使用することを想定しています。ホバー時にはボーダーと背景色が変化します。

<Tabs labels={["Preview", "Code"]}>
  <div>
    <ul style="list-style: none; padding: 0; display: flex; flex-direction: column; gap: 0.8rem; margin-bottom: 1.5rem;">
      <NavCard href="/components/button" title="Button コンポーネントを見る" />
      <NavCard href="/components/callout" title="Callout コンポーネントを見る" />
      <NavCard href="#" title="実装予定のページ" />
    </ul>
  </div>
  <div>
    ```html
    <ul style="list-style: none; padding: 0; display: flex; flex-direction: column; gap: 0.8rem;">
      <NavCard href="/components/button" title="Button コンポーネントを見る" />
      <NavCard href="/components/callout" title="Callout コンポーネントを見る" />
      <NavCard href="#" title="実装予定のページ" />
    </ul>
    ```
  </div>
</Tabs>
