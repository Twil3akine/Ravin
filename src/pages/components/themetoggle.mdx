# ThemeToggle

ライトモードとダークモードを切り替えるための専用コンポーネントです。

## Dependencies

このコンポーネントは以下の内部コンポーネントに依存しています。事前にプロジェクトへ追加してください。

<Grid>
  <NavCard href="/components/toggle" title="Toggle" />
</Grid>

## Usage

実際のサイト上でアイコン付きのトグルとして表示され、操作するとHTMLの `data-theme` 属性と `localStorage` が自動的に更新されます。

<div style="margin: 1.5rem 0; padding: 2rem; border: 1px solid var(--theme-border); border-radius: 8px; display: inline-block;">
  <ThemeToggle client:load />
</div>

<Callout type="tip" title="Tip">
  通常、このコンポーネントは記事内ではなく、`Layout.astro` の中で `Header` コンポーネントの `slot="actions"` に配置して、サイト全体で共通利用します。
</Callout>
