Ravinテーマへようこそ。この記事は `src/content/blog` ディレクトリに配置されたMDXファイルから生成されています。

## Focus on your content.

Ravinは、<Marker type="info">シンプルさとリッチな表現</Marker>を両立したAstroテーマです。

ノイズの少ないモノトーン基調のデザインが、あなたの文章を際立たせます。しかしながら、ただシンプルなだけではありません。Ravinには、記事をより魅力的で分かりやすくするための**豊富なカスタムコンポーネント**が組み込まれています。

<Callout type="note" title="ヒント">
  MDXを利用することで、Markdownの書きやすさとAstro/Svelteコンポーネントの表現力を同じファイル内でシームレスに同居させることができます。
</Callout>

## コードの表現力

技術ブログにおいて、ソースコードの読みやすさは非常に重要です。Ravinの `CodeBlock` コンポーネントを使えば、きれいにフォーマットされたコードをコピーボタン付きで配置できます。

もちろん、`Tabs` コンポーネントと組み合わせることも可能です。

<Tabs labels={["JavaScript", "Python", "Rust"]}>
  <div>
    <CodeBlock>
    ```javascript
    function greet(name) {
      console.log(`Hello, ${name}! Welcome to Ravin.`);
    }
    greet("Astro");
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```python
    def greet(name):
        print(f"Hello, {name}! Welcome to Ravin.")
        
    greet("Astro")
    ```
    </CodeBlock>
  </div>
  <div>
    <CodeBlock>
    ```rust
    fn main() {
        let name = "Astro";
        println!("Hello, {}! Welcome to Ravin.", name);
    }
    ```
    </CodeBlock>
  </div>
</Tabs>
